# E-Learning Web App

An interactive **E-Learning Platform** built with **React**, designed to help students explore courses, watch learning content, and track their progress.  
This project provides a clean and user-friendly interface for online education.

---

## Features

- Modern landing page UI
- Browse available courses
- Course preview and video learning section
- Search and filter courses
- User authentication (Sign Up / Sign In)
- User dashboard to track enrolled courses
- Protected routes (only logged-in users can access certain pages)
- Fully responsive design

---

## Tech Stack

- **React**
- **JavaScript (ES6+)**
- **CSS
